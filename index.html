Gistify

A utility to add gist creation to finder on mac.

Download this project as a .zip file Download this project as a tar.gz file

Check out Gistify

Gistify is a simple utility to post gists from text files in finder. This makes it super easy to upload some text permenantly. Here is an example of a Gistified text file. If you want to use it, just Download the source and put the service into your 

/Users/Username/Library/Services

directory. If that fails, you can try importing through automator or just paste the source into an automator shell script with the following settings:

Service receives selected files or folders in finder

pass input as arguments
